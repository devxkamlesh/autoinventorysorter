# Auto Inventory Sorter v1.0.0

## 🎉 Initial Release

A powerful client-side Minecraft mod for Fabric that automatically organizes your inventory with intelligent sorting and quality-of-life features.

## ✨ Features

### Core Features
- ✅ **One-Key Sorting** - Press R to instantly organize inventory
- ✅ **Smart Categorization** - Sort by type, rarity, name, quantity, or durability
- ✅ **Auto-Stack Merging** - Automatically combines partial stacks
- ✅ **Chest Sorting** - Sort any container with the same key

### Advanced Features
- ✅ **Hotbar Slot Locking** - Lock specific slots for weapons/tools
- ✅ **Auto-Refill System** - Automatically refills consumed items
- ✅ **Blacklist System** - Exclude items/slots from sorting
- ✅ **Profile System** - Survival, PvP, Mining, Building presets

## 📥 Installation

### Requirements
- Minecraft 1.21.8
- Fabric Loader 0.16.9+
- Fabric API 0.136.1+1.21.8
- Java 21+

### Download
1. Download `autoinventorysorter-1.0.0.jar` from releases
2. Place in `.minecraft/mods/` folder
3. Launch Minecraft!

## 🎮 Quick Start

1. Press **R** to sort your inventory
2. Press **R** in a chest to sort the chest
3. Edit config at `.minecraft/config/autoinventorysorter.json`

## 📊 Performance

- Memory: ~2MB overhead
- CPU: <0.1% average usage
- Sort time: ~5ms for full inventory

## 🐛 Known Issues

- Visual sort button disabled due to mixin compatibility (R key works perfectly)
- Some modded items may not categorize correctly
